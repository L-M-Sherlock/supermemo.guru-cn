# 优先使用经过标注的关联

原文：[Prefer labeled associations (andymatuschak.org)](https://notes.andymatuschak.org/z7pGUpz2fQsHHUPbjThz85xXPvHwrmikAeYH4)

在信息系统中连接节点的边（例如，[常青笔记之间应该建立大量链接](https://notes.andymatuschak.org/z2HUE4ABbQjUNjrNemvkTCsLa1LPDRuwh1tXC)）本身应以某种方式标注，以展现关联的背景。仅仅说「X 与 Y 有关」通常是不够的：最好说「X 在 Z 的背景下更详细地介绍了 Y。」

额外的背景将是你记住或评估链接材料的引子，这将帮助你有效地浏览知识图谱，回溯你之前的思考过程。
